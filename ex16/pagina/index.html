desenvolvimento WEB

HTML

HTML (Linguagem de Marcação de HiperTexto) é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web.

  • imagens
  • textos
  • tabelas
  • formularios

CSS

O CSS é uma linguagem de estilo usada para formatar documentos HTML. Ele define como os elementos HTML devem ser exibidos na tela, permitindo alterações como cores, fontes e espaçamento. Em resumo, o CSS torna as páginas web agradáveis e organizadas.

JavaScript

O JavaScript é uma linguagem de programação de alto nível voltada para o desenvolvimento web. Originalmente criada para funcionar nos navegadores, ela permite a criação de páginas interativas com elementos dinâmicos e boa performance1. Em resumo, o JavaScript permite implementar funcionalidades mais complexas em páginas web, como atualizações dinâmicas, controle de multimídias e animações 2D/3D
